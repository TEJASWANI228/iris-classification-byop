# Iris Flower Classification 🌸

A machine learning project that classifies iris flowers 
into 3 species using a Decision Tree classifier.

## What this project does
- Loads the Iris dataset (150 flowers, 4 features each)
- Explores and visualizes the data
- Trains a Decision Tree model
- Achieves 100% accuracy on test data
- Predicts the species of a new flower

## How to run this project

### Step 1: Install Python
Download from python.org (version 3.9 or above)

### Step 2: Install required libraries
Open terminal and run:
pip install scikit-learn pandas matplotlib

### Step 3: Run the project
python iris_project.py

## Output
- Data exploration printed in terminal
- Scatter plot saved as iris_scatter_plot.png
- Model accuracy printed in terminal
- Prediction for a sample flower

## Technologies used
- Python 3.9
- scikit-learn
- pandas
- matplotlib
